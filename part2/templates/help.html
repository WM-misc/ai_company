快速开始

欢迎使用员工绩效管理系统！本系统帮助您管理和跟踪员工绩效评估。

主要功能：
  • 查看个人绩效评估
  • 管理评估记录
  • 生成绩效报告
    • {% if session.role == 'admin' %}
  • 配置评估模板
    • {% endif %}

绩效评估

绩效评估是系统的核心功能，支持多维度的员工能力评估。

评估流程：
  1. 创建评估记录
  2. 填写评估内容
  3. 提交评估结果
  4. 生成评估报告
{% if session.role == 'admin' %}

模板管理

管理员可以配置评估模板，支持灵活的变量配置功能。

高级功能：系统支持复杂的变量配置， 包括序列化对象，适用于高级用户。
模板变量语法：
  • 标准变量：{"key": "value"}
  • 序列化变量：pickle:<encoded_data>
